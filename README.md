# Thermodynamic Illumination

Public release package for the paper and reproducibility artifacts.

## Contents

- `paper/submission_snapshot.pdf`: rendered submission PDF snapshot.
- `paper/ax.tar`: arXiv upload tarball (flat-source package).
- `paper/arxiv_flat_source/`: flattened LaTeX source used to build `ax.tar`.
- `core/`, `experiments/`, `results/`, `validation/`, `demos/`: code supplement contents.
- `REPRODUCE.md`: end-to-end reproduction notes.

## Reproduce core artifacts

See `REPRODUCE.md` for command-level instructions.

## Notes

- This repo intentionally excludes private working notes and unrelated future-paper drafts.
- Some experiments are compute-intensive; start with the smaller reproductions in `REPRODUCE.md`.
